STANDALONE TA MODULE RUNNING GUIDE

This folder is made to run directly at:

http://localhost/ta_mvc_patch_for_online_quiz_platform/login.php

Steps:
1. Replace your current folder:
   C:\xampp\htdocs\ta_mvc_patch_for_online_quiz_platform

2. Start Apache and MySQL in XAMPP.

3. Open phpMyAdmin:
   http://localhost/phpmyadmin

4. Import:
   database_patch/standalone_full_database.sql

5. Open:
   http://localhost/ta_mvc_patch_for_online_quiz_platform/login.php

6. Login:
   Email: ta@quiz.com
   Password: password
